API API Description
Earthquakes Displays data about the first generated earthquake in the json (datetime, depth and magnitude).
Oceans Displays data about the first generated ocean in the json (name and geonames id).
Weather Displays data from the most recent entry in Geonames weather dataset (Station name, temperature, cloud condition).
